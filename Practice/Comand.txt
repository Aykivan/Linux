Используемые оманды:
● sudo apt update - проверка обновлений
● sudo apt install openssh-server - Скачали ssh-server
● sudo systemctl status ssh - Запрос статуса сервера
● w - запрос пользователей 
● Ctrl + D/exit - выход
● ssh -p 8022 ai@localhost - становил соединение с виртуальной машиной
● Ctrl + C - остановка всех процессов в терминале

● ls — список файлов
● pwd — текущая директория
● mkdir — создание каталога
● cp — копирование
● rm — удаление
● mv — переименование/перенос
● touch — создание пустого файла
● cat — вывод файла, склейка, создание

● cp -v .profile profile1 - копирует файл
	-v - выводит что-то на экран
	-i - добавив это строка будет запрашивать разрешение на перезапись файла

● mv - переименовывает файл либо перемещает его

● touch имя файла - создает файл с именем (не каталог)
● cat имя файла - в основном для вывода на экран содержимого

● echo " " - выводит на экран содержимое кавычек
	-n - не переводит командную строку на следующую строку
	-e - позволяет допослять содержимое разными элементами типа \n

● ln обьект ссылка на обьект - создает жескую сетку
	-s - данный ключ создает ссылочную ссылку

 Средства просмотра:
● more - устаревший не актуальныйы, постраничный вывод 
● les - построчный вывод
	q - для выхода

 Редакторы файлов:
● vim/vi - имеет 3 режима
	i - режим редактирования
	esc - выход в командныый режим
	: - режим последней статьи w - pfgbcm q! - выйти 
	vimtutor - инфо уроки
